Step 1: Turn on your plug

  • Plug in your plug to ethernet and then into power

    • Step 2: Basic Tablet Setup

    1. Wifi: on
    2. Wifi: demo1
    3. Browser: Uncheck "Block Popup Windows"
    4. Browser: Check "Landscape-only display"

    Step 3: Install SL4A

    1. If required, go back onto the internet WIFI
    2. Install SL4a: http://code.google.com/p/android-scripting/downloads/detail?name=sl4a_r3.apk
    3. Install Sl4a python module: http://code.google.com/p/android-scripting/downloads/detail?name=python_for_android_r1.apk. Note: When you install python, you must click on the "Install" button after you launch the app

    Step 4: Install the MCP Service

    1. Remove the SD card from the tablet
    2. Plug the SD card into a desktop PC
    3. Browse to the /sdcard/sl4a/scripts directory
    4. Copy mcpservice.py (http://www.box.net/shared/kk1aummqpn ) to the sd card under /sl4a/scripts
    5. Unzip cherrypy.zip (http://www.box.net/shared/pimqpejgvm ) to the sd card under /sl4a/scripts
    6. Put the SD card back into the tablet
    7. add copy demo reset file onto SD

    Step 5: Install Additional Players

    1. Additional Players: Flash Player
    2. Install PDF reader (droidviewer: http://code.google.com/p/droidreader/downloads/detail?name=droidreader-0.5.apk&can=2&q= >
    3. Install Marvell YouTube Player (I Need a URL)

    Step 6: Launch and Run the MCP Service

    1. Open the Applications List
    2. Launch SL4A (with green eyes )
    3. To check that it is running, pull down the tooltray at the top and click on the "Click To View Running Scripts" in SL4A. There should always be one script running, mcpservice.py

    Step 7: Bookmark EpochEDU Demo on tablet

    1. Bookmark student url (list url)
    2. Bookmark teacher url
