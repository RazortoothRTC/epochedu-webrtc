To configure your system to run this demo, you should do the following:
* make sure you have perl installed
* make sure you have cpan installed
* make sure you install cpanm using cpan
* run the script: setup_epoch_edu.sh

To run the demo, you need only one system, but two or three is better for demo purposes.  A Plug computer is not stricly required.  Any web compliant browser should work.  This has been tested on:
* Firefox 3.6
* Safari 4
* Android 1.6
Ports must be available on: 5000, 5001

* run the script: run_epoch_edu.sh
